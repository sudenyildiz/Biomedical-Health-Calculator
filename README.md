# Biomedical Health & Metabolic Rate Calculator

This project is a C-based tool designed to process basic biometric data for health analysis.

## Features
- **BMI Calculation:** Categorizes according to World Health Organization (WHO) standards.
- **BMR Calculation:** Uses the Harris-Benedict equation to determine basal metabolic rate.
- **Gender-Specific Analysis:** Adjusts calculations based on biological differences.

## Technologies Used
- C Programming Language
- Code::Blocks IDE
